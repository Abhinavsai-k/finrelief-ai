# FinRelief AI Pro

## AI-Powered Debt Relief & Financial Recovery Platform

FinRelief AI Pro is an intelligent financial assistance platform that helps borrowers manage loans, analyze financial health, predict debt settlements, and generate AI-powered negotiation strategies using Google Gemini AI.

## Installation

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
